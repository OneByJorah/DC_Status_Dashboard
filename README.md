# DC Status Dashboard (Starter)

This is a **starter version** of a Domain Controller monitoring dashboard.
It is SAFE to upload publicly to GitHub.

## What this repo contains
- Flask dashboard (mock data only)
- Clean UI layout
- Public + Admin pages
- Ready for future collectors & alerts

## What this repo does NOT contain
- No credentials
- No domain names
- No secrets
- No real monitoring yet
